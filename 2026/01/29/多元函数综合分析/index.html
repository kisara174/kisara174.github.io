多元函数综合分析

多元函数微积分学习笔记

1. 二元函数泰勒公式 (Taylor’s Formula)

1.1 标准形式

设 $f(x, y)$ 在点 $(x_0, y_0)$ 附近具有 $n+1$ 阶连续偏导数，令 $h = \Delta x, k = \Delta y$，展开式为：
$$f(x_0+h, y_0+k) = f(x_0, y_0) + \sum_{m=1}^{n} \frac{1}{m!} \left( h\frac{\partial}{\partial x} + k\frac{\partial}{\partial y} \right)^m f(x_0, y_0) + R_n$$

1.2 微分算子 $(h\partial_x + k\partial_y)^m$

该算子遵循二项式展开规律，是泰勒公式的“模具”：

  • $m=1$ (一阶项)：$h f_x + k f_y$（即全微分 $df$）
  • $m=2$ (二阶项)：$h^2 f_{xx} + 2hk f_{xy} + k^2 f_{yy}$
  • 矩阵形式（二阶近似）：
    $$f(\mathbf{x}) \approx f(\mathbf{x}_0) + \nabla f(\mathbf{x}_0)^T \Delta \mathbf{x} + \frac{1}{2} \Delta \mathbf{x}^T \mathbf{H}(\mathbf{x}_0) \Delta \mathbf{x}$$
    其中 $\mathbf{H}$ 为 Hessian 矩阵（二阶偏导数组成的对称阵）。

2. 拉格朗日乘数法 (Lagrange Multipliers)

2.1 核心逻辑

用于求解条件极值。将带约束的问题转化为无约束问题。

  • 目标函数：$f(x, y)$
  • 约束条件：$g(x, y) = 0$
  • 构造拉格朗日函数：$L(x, y, \lambda) = f(x, y) + \lambda \cdot g(x, y)$
  • 求解：通过方程组 $\nabla L = 0$（即对 $x, y, \lambda$ 各自求偏导并令其为 0）找到极值候选点。

2.2 典型应用：证明 AM-GM 不等式

题目：证明 $\sqrt[n]{x_1 x_2 \dots x_n} \leq \frac{x_1 + x_2 + \dots + x_n}{n}$
证明思路：

  1. 设定约束 $\sum_{i=1}^n x_i = S$（定值）。
  2. 构造 $L(x_1, \dots, x_n, \lambda) = \prod_{i=1}^n x_i + \lambda(\sum_{i=1}^n x_i - S)$。
  3. 对各 $x_i$ 求偏导并令其为 0，得到 $x_1 = x_2 = \dots = x_n = S/n$ 时乘积取得最大值。
  4. 由此推导出算术平均值 $\geq$ 几何平均值。

3. 数量场与方向导数

3.1 数量场 (Scalar Field)

  • 概念：空间中每个点对应一个数值（如温度 $T$、气压 $P$、海拔 $H$）。
  • 几何体现：等值线（2D）或等值面（3D）。在这些线上，函数值保持不变。

3.2 方向导数 (Directional Derivative)

  • 定义：函数沿任意指定方向 $\boldsymbol{l}$ 的变化率。
  • 计算公式：
    $$\frac{\partial f}{\partial l} = f_x \cos\alpha + f_y \cos\beta + f_z \cos\gamma$$
    (其中 $\cos\alpha, \cos\beta, \cos\gamma$ 是方向向量的单位方向余弦)
  • 推导逻辑：利用一阶泰勒展开 $\Delta f \approx \frac{\partial f}{\partial x}\Delta x + \frac{\partial f}{\partial y}\Delta y + \frac{\partial f}{\partial z}\Delta z$，其中 $\Delta x = \rho \cos\alpha$ 等，代入定义式求极限即得。

4. 梯度 (Gradient)

4.1 数学定义

梯度是一个向量，由函数在某点处的所有一阶偏导数组成：
$$\nabla f = \text{grad} f = \left( \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z} \right)$$

4.2 核心性质

  1. 增长最快：梯度向量指向函数值增长最快的方向。
  2. 模长意义：梯度的模 $|\nabla f|$ 是所有方向导数中的最大值。
  3. 几何关系：梯度向量始终垂直于过该点的等值面/等值线（是其法向量）。
  4. 与方向导数的投影关系：
    $$\frac{\partial f}{\partial l} = \nabla f \cdot \mathbf{e}_l = |\nabla f| \cos\theta$$
    • 当方向 $l$ 与梯度方向一致 ($\theta=0$)，变化率最大。
    • 当方向 $l$ 与梯度方向垂直 ($\theta=90^\circ$)，变化率为 0。
